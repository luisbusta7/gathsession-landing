# ConquerBlocks CSS Homework 2 - GathSession Landing Page

Landing page exercise built as part of the ConquerBlocks CSS homework.  
The project recreates a modern community platform hero section using semantic HTML, Sass architecture, and Vite as the development environment.

## 📌 Overview

This project focuses on building a clean and visually structured landing page based on a provided design reference.

The layout includes:

- A dark themed header
- Navigation menu
- Large hero section
- Main call-to-action buttons
- Feature highlight cards
- Decorative image composition
- Sass partial-based architecture
